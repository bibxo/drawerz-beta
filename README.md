# Drawerz Beta - Web Animation Sketching Tool

Drawerz is an interactive web-based sketching and animation tool that allows users to create, animate, and export animated drawings. This versatile tool is designed for artists, animators, and creative enthusiasts looking to quickly create animated sketches with minimal setup.

## Features

### Drawing Tools
- **Pen Tool**: Create smooth strokes with customizable color and size
- **Eraser Tool**: Easily correct mistakes
- **Adjustable Stroke Size**: Fine-tune your drawing with precise control

### Layer Management
- **Multiple Layers**: Organize your artwork with separate layers
- **Layer Controls**: Reorder, rename, show/hide, and adjust opacity
- **Layer-specific Animation**: Apply different animation effects to each layer

### Animation Effects
- **Wiggle Intensity**: Add organic movement to your strokes
- **Breathing Stroke**: Create pulsating line thickness effects
- **Shake Intensity**: Add dynamic vibration to entire layers
- **Animation Speed**: Control the tempo of all animations

### Project Management
- **Save/Load Projects**: Store your work in .drz format for later editing
- **Export as Video**: Generate MP4/WebM videos of your animations
- **Undo/Redo**: Easily backtrack or restore changes

### User Interface
- **Responsive Design**: Works on desktop and mobile devices
- **Collapsible Control Panel**: Maximize your drawing space when needed
- **Keyboard Shortcuts**: Boost productivity with hotkeys

## Getting Started

### Quick Start
1. Open `index.html` in a modern web browser
2. Select a drawing tool (Pen/Eraser)
3. Adjust the stroke size and color
4. Draw on the canvas
5. Add animation effects using the sliders
6. Save your work or export as video

### Keyboard Shortcuts
- **Pen Tool**: P
- **Eraser Tool**: E
- **Save Project**: Ctrl/Cmd + S
- **Load Project**: Ctrl/Cmd + O
- **Export Video**: Ctrl/Cmd + E
- **Undo**: Ctrl/Cmd + Z
- **Redo**: Ctrl/Cmd + Y
- **Clear All Layers**: Ctrl/Cmd + Shift + Delete

## Technical Details

### Browser Compatibility
Drawerz is built with modern web technologies and works best in the latest versions of:
- Chrome
- Firefox
- Edge
- Safari

### Dependencies
- Font Awesome (6.0.0) for icons
- Google Fonts (Montserrat) for typography
- No JavaScript frameworks - built with vanilla JavaScript

### File Format
The `.drz` file format is a JSON-based format that stores:
- All drawing strokes
- Layer information
- Animation settings

## Installation

No installation required! Simply download or clone this repository and open `index.html` in your web browser.

## Hosting Your Own Instance

To host Drawerz on your own web server:

1. Upload all files to your web hosting service
2. Ensure the server supports static HTML/CSS/JavaScript files
3. No database or server-side code required
